# Explosion Vent Sizing

Internal tool used by engineering and sales teams to appropriately size and
report on various explosion vents. Accepts a wide set of parameters and performs
standards (NFPA) based calculations. Role based administrative systems allow
lead engineers to handle standards updates with code changes
